## github.io page

See [https://jschaul.github.io](https://jschaul.github.io)

## Usage

- ensure ruby >= 2.0 (check with `ruby --version`).
    - install [rvm](https://rvm.io/)
    - `source ~/.rvm/scripts/rvm`
    - `rvm install 2.3`
- `gem install jekyll`
- `jekyll serve`

See preview under [http://127.0.0.1:4000](http://127.0.0.1:4000)
